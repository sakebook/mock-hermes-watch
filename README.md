### Local
- Node.js 8.+

## Build
### Local

```sh
$ git clone git@github.com:sakebook/mock-hermes-watch.git
$ cd mock-hermes-watch
mock-hermes-watch$ npm install
mock-hermes-watch$ npx fitbit
fitbit$ build
fitbit$ install
```

### Fitbit Studio
```sh
$ git clone git@github.com:sakebook/mock-hermes-watch.git
```

and D&D `app/`, `resources/`, `package.json`
